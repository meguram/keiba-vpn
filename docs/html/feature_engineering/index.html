カテゴリ早見

1. レース内圧力場

  • 指数の field 内分散・IQR（混戦度）
  • 能力分布のエントロピー

2. 馬・厩舎・騎手のリズム

  • 同表面からの日数・連闘圧力
  • 厩舎着順ボラティリティ

3. コース・馬場

  • 開催内クッション Z、重馬場父系統率

4. ペース・位置のルール近似

  • 逃げ・先行の「前圧」合成、枠×脚質

5. race_performance（過去のみ）

  • 同一距離帯での移動平均、対戦強度履歴

6. メタ特徴

  • 参照できた過去走本数、指数有無フラグ

検証と記録

  • 改善時は feature_dictionary.html / _registry.json 更新。
  • MLflow の run_id を必ず紐付け（modeling_coding_strategy.html）。
