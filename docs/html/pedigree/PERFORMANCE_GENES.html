競走馬パフォーマンス遺伝子ナレッジベース

競走馬の運動能力に関係する遺伝子マーカーの知識ベースと推論システムです。遺伝子情報は「どの距離・条件に向いているか」を客観的に推定する手がかりになるため、血統適性の評価を補完する特徴量として活用します。

対象とする遺伝子マーカー

現在、以下の遺伝子マーカーをカバーしています。

遺伝子マーカー別名・分類機能・競走への影響
MSTN Myostatin・ミオスタチン遺伝子 筋肉量を制御。距離適性に最も影響する。筋肉質な馬はスプリント向き、スリムな馬は長距離向き
DMRT3 歩様遺伝子 ギャロップ（通常の競走走法）か側対歩・速歩かを決める。サラブレッドはほぼCC型で固定
PDK4 エネルギー代謝遺伝子 糖（グルコース）か脂肪酸どちらを優先的に使うかを決める。長距離では脂肪酸酸化優位（AA型）が有利
COX4I2 ミトコンドリア機能遺伝子 酸素の利用効率と持久力に関係する
BIEC2-808543 距離適性SNPマーカー MSTN座近傍の多型マーカー。MSTNと組み合わせると距離予測精度が向上する
その他候補遺伝子 研究段階 —

データファイル

ファイル内容
data/knowledge/myostatin_genes.json ミオスタチン遺伝子（MSTN）専用のナレッジベース。130頭以上の競走馬・種牡馬の遺伝子型データ。産駒逆推論による継続的更新
data/knowledge/performance_genes.json MSTN以外のパフォーマンス関連遺伝子。遺伝子機能・多型・競走能力との関連・遺伝子間相互作用を記録

Pythonモジュール

src/research/myostatin.py

ミオスタチン遺伝子専用の解析モジュール。

from research.genes.myostatin import MyostatinLookup

mstn = MyostatinLookup()

# 馬の遺伝子型を検索（種牡馬・競走馬どちらも対象）
info = mstn.get_sire_info("ディープインパクト")
# → {'genotype': 'TT', 'confidence': 'confirmed', ...}

# 産駒の遺伝子型確率を推定
probs = mstn.predict_offspring("ディープインパクト", "キングカメハメハ")
# → {'CC': 0.0, 'CT': 0.5, 'TT': 0.5}

# 距離適合度を算出
affinity = mstn.distance_affinity("ディープインパクト", "キングカメハメハ", 2400)
# → 0.8547（高適合）

src/research/performance_genes.py

MSTN以外のパフォーマンス遺伝子解析モジュール。

from research.genes.performance_genes import PerformanceGenes

pg = PerformanceGenes()

# DMRT3（歩様）の適合度
gait = pg.dmrt3_gait_type("CC")  # → "gallop"
suitability = pg.dmrt3_thoroughbred_suitability("CC")  # → 1.0

# PDK4（代謝）の距離適性
profile = pg.pdk4_distance_preference("AA")
# → {'optimal_distance': 2200, 'stamina_score': 0.9, ...}

# 複数遺伝子の統合プロファイル
combined = pg.get_combined_distance_profile(
    mstn="CT",
    pdk4="GA",
    biec2="CT",
    distance_m=1800
)
# → {
#     'optimal_distance': 1750,
#     'distance_score': 0.95,
#     'speed_index': 0.5,
#     'stamina_index': 0.5,
#     ...
# }

# 極端表現型（超スプリント/超ステイヤー）の判定
extreme = pg.check_extreme_phenotype(mstn="TT", pdk4="AA")
# → {'is_extreme': True, 'phenotype': 'ultra_stayer', ...}

CLI使用例

ミオスタチン遺伝子

# 馬の遺伝子型を検索（種牡馬・競走馬どちらも対象）
python3 -m src.research.genes.myostatin lookup ディープインパクト

# 産駒の遺伝子型確率を推定
python3 -m src.research.genes.myostatin predict ディープインパクト キングカメハメハ --distance 2000

# 登録馬一覧（種牡馬・競走馬）
python3 -m src.research.genes.myostatin list

# 統計情報
python3 -m src.research.genes.myostatin stats

その他のパフォーマンス遺伝子

# 登録遺伝子一覧
python3 -m src.research.genes.performance_genes list

# 遺伝子情報表示
python3 -m src.research.genes.performance_genes gene DMRT3

# DMRT3遺伝子型の評価
python3 -m src.research.genes.performance_genes dmrt3 CC

# PDK4遺伝子型の距離適性
python3 -m src.research.genes.performance_genes pdk4 AA

# 統合的距離適性プロファイル
python3 -m src.research.genes.performance_genes combined \
  --mstn CT --pdk4 GA --biec2 CT --distance 2000

# 極端表現型チェック
python3 -m src.research.genes.performance_genes extreme --mstn TT --pdk4 AA

パイプラインへの組み込み

src/pipeline/feature_builder.py

特徴量生成時に、遺伝子関連の特徴量が自動で追加されます。

from research.genes.myostatin import get_lookup as get_mstn
from research.genes.performance_genes import get_lookup as get_perf_genes

mstn = get_mstn()
pg = get_perf_genes()

# ミオスタチン特徴量（既存）
mstn_features = mstn.offspring_features(
    sire_name="ディープインパクト",
    dam_sire_name="キングカメハメハ",
    distance_m=2000
)
# → {
#     'sire_mstn_c': 0.0, 'sire_mstn_t': 1.0,
#     'mstn_cc_prob': 0.0, 'mstn_ct_prob': 0.5, 'mstn_tt_prob': 0.5,
#     'mstn_distance_affinity': 0.85,
#     'mstn_speed_index': 0.0,
#     ...
# }

# 統合的距離プロファイル（新規）
# ※種牡馬のPDK4/BIEC2遺伝子型データが蓄積されたら利用可能
combined_profile = pg.get_combined_distance_profile(
    mstn="TT",  # MSTNはmstn.predict_offspringから取得
    pdk4="GA",  # 将来的に種牡馬DBから取得
    biec2="CT",
    distance_m=2000
)
# → {
#     'optimal_distance': 2100,
#     'distance_score': 0.92,
#     'speed_index': 0.3,
#     'stamina_index': 0.7,
#     ...
# }

生成される主な特徴量:

特徴量名説明
mstn_cc_prob, mstn_ct_prob, mstn_tt_prob MSTN遺伝子型（CC/CT/TT）の推定確率
mstn_distance_affinity MSTN単独で計算した距離適合度
mstn_speed_index スプリント寄りの度合い（0=長距離寄り, 1=短距離寄り）
gene_optimal_distance 複数遺伝子を統合した最適距離の推定値
gene_distance_score 対象レース距離での遺伝子的な適合度
gene_speed_index 統合スピード指数
gene_stamina_index 統合スタミナ指数

遺伝子型の推定方法

遺伝子型データの取得方法は信頼度によって3段階あります。

直接データ（信頼度：高）

  1. 一口馬主クラブ・@Plucky_Liege 等が公表した遺伝子検査結果
  2. 学術論文での個体遺伝子型の報告
  3. Equinome・LRC等の商業検査サービスのデータ

逆推論（信頼度：中）

産駒の遺伝子型が判明している場合、メンデル遺伝の法則から親の型を絞り込めます。

産駒の遺伝子型推論できる親の制約
CC型確定（例: サトノカルナバル） 父（例: キタサンブラック）はCアレルを必ず持つ（CC or CT）
CC型が確認（例: ゴールドシップ産駒） 親（例: ゴールドシップ）はCT推定（TT型は除外される）
TT型の産駒が誕生 両親ともTアレルを保有（CC型は除外）
CC型の産駒が誕生 両親ともCアレルを保有（TT型は除外）

血統推論（信頼度：低〜中）

  1. 父の遺伝子型から産駒の型を統計的に推定
  2. 同系統馬の遺伝子型傾向を参考にする
  3. 競走成績・産駒傾向からの間接的な推定

データ更新の手順

1. 新規情報の収集

  • X/Twitter（@Plucky_Liege 等）での産駒遺伝子型の報告
  • 一口馬主クラブの検査結果公表
  • 学術論文・業界誌の新知見

2. 逆推論の実行

from research.genes.myostatin import MyostatinLookup

mstn = MyostatinLookup()

# 産駒の遺伝子型から親の制約を推論
result = mstn.reverse_infer(
    offspring_genotype="CC",
    sire_name="キタサンブラック"
)
# → {
#     'sire_constraints': {
#         'has_c': True,
#         'possible_genotypes': ['CC', 'CT'],
#         'excludes': ['TT']
#     },
#     ...
# }

3. ナレッジベースの更新

  • myostatin_genes.json を直接編集します
  • 種牡馬だけでなく、主要な競走馬（GI勝ち馬など）も記録します
  • confidence フィールドに信頼度レベルを適切に設定します
  • source フィールドに情報の出典を記録します
  • update_log に変更履歴を追記します

4. 検証

# 整合性チェック（遺伝法則に矛盾がないか）
python3 -m src.research.genes.myostatin stats

# 産駒推定のテスト
python3 -m src.research.genes.myostatin predict <父名> <母父名>

遺伝子間の相互作用

組み合わせ遺伝的関係予測への影響
MSTN × PDK4
TT × AA		 独立して作用 超長距離型（2400m以上向き）。スリムな体型＋脂肪酸酸化優位で極長距離に特化しやすい
MSTN × PDK4
CC × GG		 独立して作用 超短距離型（1000〜1200m向き）。筋肉質の体型＋グルコース代謝優位で極短距離に特化しやすい
MSTN × BIEC2-808543 染色体上で近い位置にあるため同時に遺伝しやすい（連鎖不平衡） 両マーカーを組み合わせると単独より距離適性の予測精度が向上する
DMRT3 × MSTN 異なる染色体に位置するため独立して遺伝する DMRT3は歩様パターン、MSTNは筋量と距離適性を別々に制御。サラブレッドではDMRT3=CCがほぼ固定されているため実質的な差異はほとんどない

限界と注意事項

#注意点詳細
1 競走能力は多数の遺伝子が関与する 競走能力は数百〜数千の遺伝子が絡む複雑な形質。少数マーカーだけで能力を説明できる範囲は限定的
2 環境要因の影響が大きい 調教内容・栄養管理・騎手の技術・馬場状態などの環境要因は、遺伝要因と同等かそれ以上に競走結果に影響する
3 遺伝子型は「傾向」であり「確定」ではない 同じ遺伝子型を持っていても個体差は大きい。「この型なら必ず長距離が得意」とはならない
4 データに偏りがある GI勝ち馬・種牡馬などエリート馬のデータは豊富だが一般競走馬は不足。日本馬が中心で欧米馬のデータは限られる。遺伝子検査は一口馬主クラブの馬が主体で代表性に限界あり
5 未発見の遺伝子が多数ある 現在把握しているマーカーは氷山の一角。多くの重要遺伝子がまだ発見されていない
