📋 このセクションについて
血統適性プロファイルは、レース結果データの集計から生成されています。 日々レース結果が追加されていくため、定期的に再計算することで精度が向上します。
  • 再生成 (Rebuild): アーティファクトを最新データで再計算 (バックグラウンド実行)
  • リロード (Reload): 既存ファイルを API キャッシュに再読込 (高速・無害)
  • 状態 (Status): 各アーティファクトの最終更新日時とサイズを表示
⚠ 開発者特権: このメニューは /api/admin/ 配下の API を使うため、ログイン認証 (DEV_PASSWORD) が必要です。 一般ユーザーには非表示。再生成は重い処理 (5秒〜数分) のため運用時は注意。
